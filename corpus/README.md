# Collective Corpus

Shared knowledge base that the toolkit draws from. Content curated in collaboration with Working Group #2 (Data).

These directories are placeholders — content will be added as the Data working group curates materials.

| Directory | Contents |
|-----------|----------|
| `guidelines/` | Accessibility standards (WCAG, ARIA, platform-specific) |
| `best-practices/` | Patterns and recommendations |
| `personas/` | User personas for different ability profiles and contexts |
| `benchmarks/` | Evaluation criteria and test suites |
